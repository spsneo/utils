 ## Command line tools
 
 1. [fkill](https://github.com/sindresorhus/fkill-cli) - Awesome kill utility
 2. [blush](https://github.com/arsham/blush) - Alternate to grep with colors
 3. [gron](https://github.com/tomnomnom/gron) - JSON grep utility
